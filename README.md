# yongge-skill

> 勇哥说餐饮的思维框架与表达方式 Skill

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![skills.sh Compatible](https://img.shields.io/badge/skills.sh-Compatible-green)](https://skills.sh)

把「勇哥说餐饮」蒸馏成一个可直接调用的 Skill，用来分析餐饮创业、门店经营、加盟风险、止损决策与小微门店生存问题。

这个 Skill 基于公开内容调研整理，重点不在鸡血和故事，而在现实取证、现金流、生存权、低杠杆试错。

## 安装

```bash
npx skills add SeanSkills/yongge-skill
```

如果你是手动使用，也可以直接查看仓库里的 [SKILL.md](./SKILL.md)。

## 适用场景

- 想开餐饮店，但不确定项目能不能活
- 已经开店，想判断还有没有继续做的必要
- 评估加盟、快招、选址、商圈和成本结构风险
- 需要用更强现实感的方式审视餐饮创业计划

## 典型触发词

- 用勇哥的视角
- 勇哥说餐饮
- 勇哥餐饮创业说
- 餐饮勇哥视角
- 如果勇哥会怎么做

## 这个 Skill 会怎么回答

- 直接用第一人称进入角色
- 先问流水、毛利、房租、人工、客流、品类、位置、负债
- 先判断有没有生存权，再谈增长和优化
- 强调现场证据，不鼓励空泛判断
- 对加盟和快招风险保持高警惕

## 使用提醒

- 这个 Skill 更适合小微线下餐饮、门店经营和加盟评估
- 不适合替代法律、财务、税务等专业意见
- 调研结论基于公开资料，人物表达和平台内容存在表演化边界
