除了张雪峰老师外，抖音还有哪位大名鼎鼎的灵魂导师？<br/><br/>没错，就是餐饮创业导师——勇哥<br/><br/>经过调试，我觉得赛博勇哥已经有九成勇哥味了<br/><br/>唯一的遗憾，就是没有360度转一圈<br/><br/>不过作为连麦不到勇哥的代餐，或者只是用来验证想法<br/><br/>或者就是图一乐<br/><br/>我相信赛博勇哥不会让你失望

## 安装

```bash
npx skills add SeanSkills/yongge-skill
```

如果你是手动使用，也可以直接查看仓库里的 [SKILL.md](./SKILL.md)。

## 适用场景

- 想开餐饮店，但不确定项目能不能活
- 已经开店，想判断还有没有继续做的必要
- 评估加盟、快招、选址、商圈和成本结构风险
- 需要用更强现实感的方式审视餐饮创业计划

## 典型触发词

- 用勇哥的视角
- 勇哥说餐饮
- 勇哥餐饮创业说
- 餐饮勇哥视角
- 如果勇哥会怎么做

## 这个 Skill 会怎么回答

- 直接用第一人称进入角色
- 先问流水、毛利、房租、人工、客流、品类、位置、负债
- 先判断有没有生存权，再谈增长和优化
- 强调现场证据，不鼓励空泛判断
- 对加盟和快招风险保持高警惕

## 使用提醒

- 这个 Skill 更适合小微线下餐饮、门店经营和加盟评估
- 不适合替代法律、财务、税务等专业意见
- 调研结论基于公开资料，人物表达和平台内容存在表演化边界
